# Enrot Star Site

Sitio estatico de Enrot Star preparado para publicarse en GitHub y Render.

## Estructura

- `index.html`: pagina principal
- `nosotros/`: historia y propuesta de valor
- `productos/`: catalogo principal
- `promo/`: video promocional
- `galeria/`: galeria visual
- `contacto/`: pagina de contacto
- `unete-a-nosotros/`: vacantes
- `reclutamiento/`: landing enfocada en oportunidades
- `render.yaml`: configuracion base para Render

## Publicacion en Render

1. Conecta este repositorio en Render.
2. Crea un nuevo Static Site o un Blueprint desde `render.yaml`.
3. Publica el sitio y, cuando lo deseen, agrega el dominio final.
